**Задание для соискателя на должность разработчика Отдела Разработки ПО, ООО НПП ТИК.**

**Создать десктоп приложение, позволяющее генерировать сигналы по заданному закону, хранить их и просматривать.**

**Функциональные требования**

**1. Генерация сигналов**

- Реализовать возможность генерации сигналов следующих типов:

- Синусоида (параметры: амплитуда, частота, фаза, количество точек)
- Меандр (параметры: амплитуда, частота, количество точек)
- Треугольный сигнал (параметры: амплитуда, частота, количество точек)
- Пилообразный сигнал (параметры: амплитуда, частота, количество точок)

- Возможность задания временного интервала генерации
- Валидация входных параметров
- Возможность задать количество точек при генерации

**2. Хранение данных (SQLite)**

- Сохранение сгенерированных сигналов в базу данных SQLite
- Структура БД должна включать:

- Метаданные сигнала (тип, параметры, дата создания)
- Точки сигнала (время, значение)

- Возможность загрузки ранее сохраненных сигналов
- История генерации с возможностью просмотра
- Возможность удалять сигналы

**3. Визуализация**

- ==Отображение графика сигнала в реальном времени после генерации
- Использование бесплатной библиотеки для построения графиков (например: LiveCharts)
- ==Возможность масштабирования и навигации по графику
- ==Отображение параметров текущего сигнала

**4. Интерфейс (WPF)**

- Форма ввода параметров генерации с понятными полями
- Область визуализации графика
- Список сохраненных сигналов с возможностью выбора и отображения
- Кнопки управления: "Генерировать", "Сохранить", "Загрузить", =="Очистить"

**Технические требования**

- ==Язык: C# (.NET 9)
- UI: WPF
- База данных: SQLite
- Библиотека графиков: любая бесплатная (с указанием выбора)

**Критерии оценки**

- Корректность работы генерации сигналов
- Правильная работа с БД (сохранение/загрузка)
- Качество визуализации
- Архитектура приложения и читаемость кода
- Обработка ошибок и валидация входных данных
- Удобство интерфейса

- Возможность наложения нескольких сигналов на один график
- Применение математических операций к сигналу
- Unit-тесты для генераторов сигналов
